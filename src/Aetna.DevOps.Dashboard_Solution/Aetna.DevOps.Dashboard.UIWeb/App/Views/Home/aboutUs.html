About the DevOps Dashboard

Use this page to get an overview of this application.

This website was developed to give an overview and summary of the DevOps pipeline and tools. This application was developed by Web Engineering/Platform Engineering in collaboration with CCSU Sutdents in Fall 2017, using the following technologies:

  •  - AngularJS for client-side routing, AJAX calls, and UI data binding.
  •  - Bootstrap for the look/feel and for a mobile-friendly, responsive design.
  •  - ASP.NET Web API for the REST service backend, caching, and for secure access to CouchDB.
